# 🔐 Security

## 📣 Reporting

If you believe you have found a security issue in a repository built from this boilerplate, report it through the repository's private security reporting channel when available.

If private reporting is not available, contact the repository maintainers directly and avoid public disclosure until the issue has been reviewed.

See [SECURITY_POLICY.md](./docs/policy/SECURITY_POLICY.md) for the operating baseline that applies to source control, credentials, access control, dependency hygiene, and CI/CD security.
