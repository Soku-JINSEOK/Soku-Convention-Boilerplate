# Boilerplate v1.0.0

Release axis: boilerplate

Catalog contracts: core-v1 and index-v2

Profiles: bootstrap, standard (default and legacy), and scaled

Compatible soku: v0.1.0

Migration from previous release: none (first public release)

Provider compatibility: provider-v1

Companion tag: soku/v0.1.0

This is the first public convention baseline. It publishes the `core-v1`
catalog with the `index-v2` profile composition contract. Sources without the
index remain readable as legacy `standard` profile sources. There is no prior
public release to migrate from, so no file removals, ownership transitions,
mergeable-path changes, or manual forward-migration actions apply.

The release is lifecycle-tested with `soku v0.1.0` and provider-v1. Compatibility
outside the versions named here is not implied.
